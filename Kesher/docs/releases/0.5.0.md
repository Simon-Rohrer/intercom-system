# Kesher 0.5.0

Release date: 2026-03-26

## Highlights

- Added role-first Companion integration with dedicated discovery and bridge websocket flows.
- Introduced image stream bridge support in backend and UI integration paths.
- Expanded admin and user interfaces for Companion profile management and Stream Deck configuration.

## New Features

- Companion profile and role-based binding model with improved state handling and command routing.
- New Companion-focused backend tests and frontend admin panels.
- Image stream backend API and integration example.
- User-facing Companion settings components and enhanced session handling.

## Improvements

- Updated backend routing/state models for Companion and realtime behavior.
- Improved frontend API typing and intercom session hooks.
- Expanded test coverage across backend companion logic and frontend Companion UI paths.
- Configuration and documentation updates for Companion setup and image stream bridge workflows.

## Notable Documentation

- Added Companion setup guide.
- Added image stream bridge documentation.

## Migration / Upgrade Notes

- Companion endpoints are roleId-first. Legacy username query parameters are no longer supported in the Companion websocket bridge path.
- Review deployment/config examples before rollout if Companion or image stream features are enabled.
