# Kesher 0.7.10

Release date: 2026-04-02

## Highlights

- Added local audio test loopback controls in sound settings so operators can validate microphone/device setup before going live.
- Improved desktop audio engine handling for more reliable local and session audio behavior.
- Extended backend auth/session configuration with stronger defaults and coverage in tests.
- Refined station/intercom settings UX and consistency across collapsible sections.

## Changes

### Frontend / Client Core

- Added local mic loopback testing flow and supporting settings/state wiring.
- Updated station intercom and connection setup UI styling and behavior.
- Expanded settings model and tests for new audio and session-related options.

### Desktop

- Updated Tauri audio engine integration to better support loopback and device flow consistency.

### Backend

- Enhanced auth/session handling and configuration wiring.
- Added/updated tests for auth and config behavior.

## Migration / Upgrade Notes

- No database migration required.
- Existing deployments can upgrade directly to `v0.7.10`.
