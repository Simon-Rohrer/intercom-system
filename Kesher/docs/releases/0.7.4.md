# Kesher 0.7.4

Release date: 2026-04-01

## Highlights

- Added release build version injection from GitHub tags in CI, so published binaries carry the correct semantic version.
- Improved the admin routing matrix usability with a horizontal scroll wrapper for narrower viewports.
- Introduced persistent network settings in the desktop app to keep operator connectivity configuration across restarts.
- Fixed a WebRTC readiness race and aligned username identity consistency in signaling and session flows.

## Changes

### CI/CD

- `release-binaries` workflow now injects the GitHub tag into build metadata/version output.

### Admin UI

- Added horizontal scrolling behavior around the routing matrix to prevent clipping and improve access on smaller screens.

### Desktop

- Added persistent network settings menu support in desktop, improving reliability for repeated startup/use.

### Realtime / Identity

- Resolved WebRTC startup ordering race conditions that could delay or block ready state.
- Standardized username identity handling to reduce mismatch issues across connected components.

## Migration / Upgrade Notes

- No database migration required.
- Existing deployments can upgrade directly to `v0.7.4`.
